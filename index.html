Cài đặt Firebase

Thực hiện một lần duy nhất — miễn phí vĩnh viễn

  1. Vào console.firebase.google.com → đăng nhập Google
  2. Nhấn Add project → đặt tên → tắt Analytics → Create project
  3. Trong project: nhấn biểu tượng </> (Web) → đặt tên app → Register app
  4. Copy đoạn firebaseConfig hiện ra
  5. Mở file family-blog-firebase.html bằng Notepad → tìm FIREBASE_CONFIG → thay các giá trị
  6. Quay lại Firebase Console → Firestore Database → Create database → Start in test mode → Next → Enable
  7. Upload lại file HTML lên GitHub → website hoạt động đầy đủ!
💡 Gói miễn phí Firebase Spark: 1GB lưu trữ, 50.000 lượt đọc/ngày, 20.000 lượt ghi/ngày — đủ dùng cho cả gia đình nhiều năm.
